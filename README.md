# Agni's Bubble Shooter

A classic and polished Bubble Shooter game where you shoot colored bubbles to match and pop them.

## How to Play

1.  **Open `index.html` in a web browser.**
    *   You can do this by double-clicking the `index.html` file in the project folder.

2.  **Controls:**
    *   **Desktop:** Aim with your mouse and click to shoot.
    *   **Mobile:** Aim with your finger and tap to shoot.

3.  **Goal:**
    *   Match 3 or more bubbles of the same color to pop them.
    *   Clear all the bubbles from the screen to win.
    *   The game is over if the bubbles reach the bottom of the screen.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: Contains all the styles for the game.
*   `script.js`: Contains all the game's logic.

## Notes

*   This is a complete Bubble Shooter game with a colorful and attractive design.
*   The game is responsive and should work on both desktop and mobile devices.
